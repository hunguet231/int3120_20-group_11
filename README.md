## Công nghệ sử dụng:

- React Native (Expo)
- Amazon Web Services

## Môi trường:

- Node.js LTS
- Visual Studio Code
- Expo App

## Cách chạy:

1. Install dependencies

```sh
$ npm install
```

2. Pull backend

```sh
$ amplify pull --appId d15pniii3p3vew --envName dev
```

Sau đó dùng thông tin được cung cấp trong file credentials.csv để thực hiện pull backend.

3. Run project

```sh
$ npm start
```
